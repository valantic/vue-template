#### `default`

Always use **v-model** to bind value to form elements!

```
<template>
  <div>
  
    <e-input v-model="demo" name="demo" placeholder="enter text..." title="foo"/> 
    
    <div v-if="demo" class="spacing--top-15">
      <p>{{demo}}</p>
    </div>
  
  </div>
</template>

<script>
  export default {
    name: 'example',
    data: () => ({
      demo: ''
    })
  };
</script>
```
