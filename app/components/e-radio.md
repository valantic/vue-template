### e-radio

Always use **v-model** to bind value to form elements!

```vue
<template>
  <div>
    <e-radio id="radio-1" 
             name="example-radio-1" 
             value="radio-1"
             v-model="demo"
             displayName="Select Radio Button"
    />
  
    <div class="spacing--top-15">
      <p>{{ demo }}</p>
    </div>
  </div>
</template>

<script>
  export default {
    name: 'example',
    data: () => ({
      demo: ''
    })
  };
</script>
```
